# Cat-Generator-Bot
Bot made with the Discord Developer API that generates a Cat when different commands are invoked.
The commands are the following:
```bash
  ;i : for the instructions.
  
  ;cat : for a cat image.
  
  ;gif : for a cat gif.
  
  ;category : for a specific cat category.
```
